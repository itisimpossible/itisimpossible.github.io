Bus Architecture

What is Bus Architecture?

A bus architecture defines how components in a computer system are interconnected and communicate with each other. It's the backbone of data transfer within a computer.

Common Bus Architectures

Single Bus

All devices share one common bus. Simple but can cause bottlenecks.

Multiple Bus

Separate buses for different functions (memory bus, I/O bus). Improves performance.

Hierarchical Bus

Buses arranged in hierarchy with bridges connecting them.

Types of Buses

Three Main Bus Types

Data Bus

Function: Carries actual data between components

Direction: Bidirectional

Width: Determines word size (32-bit, 64-bit)

Address Bus

Function: Carries memory addresses

Direction: Unidirectional (CPU to memory)

Width: Determines addressable memory

Control Bus

Function: Carries control signals

Direction: Bidirectional

Signals: Read, Write, Interrupt, Clock
