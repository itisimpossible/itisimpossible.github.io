ROM Working Principles

1. Address Input: CPU sends memory address via address bus

2. Address Decoding: ROM decoder converts address to select specific word line

3. Word Line Activation: Selected row activates all memory cells in that row

4. Bit Line Sensing: Sense amplifiers detect which bits are stored (1 or 0)

5. Data Output: Retrieved data sent via data bus to CPU
