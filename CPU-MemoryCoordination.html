How CPU Coordinates with Memory

Fetch Cycle: The CPU retrieves instructions from memory. The Program Counter (PC) holds the address of the next instruction. This address is copied to the Memory Address Register (MAR), and the instruction is fetched into the Memory Data Register (MDR), then transferred to the Instruction Register (IR).

Execute Cycle: The CPU decodes and executes the instruction. If the instruction requires data from memory, the CPU sends the address to MAR, reads the data into MDR, and processes it using the ALU. For store operations, data is placed in MDR, the address in MAR, and then written to memory.

This simulator demonstrates this coordination step by step, highlighting the relevant registers and memory locations during each operation.
